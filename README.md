# task-manager
A web-based Task Manager built with HTML, CSS, and JavaScript that allows users to manage tasks and projects with features like filtering, sorting, prioritization, and dark mode. Data is stored in the browser using localStorage, making it a lightweight and responsive solution for personal task management.

Features

-Task Management

-Add, edit, and delete tasks

-Assign tasks to projects

-Add labels and descriptions

-Mark tasks as completed or important

-Project Management

-Create and delete projects

-Assign tasks to specific projects

-Color-coded projects for easy organization

-Filtering and Sorting

-Filter tasks by priority (High, Medium, Low)

-Filter tasks by due date (Today, This Week, This Month, Overdue)

-Search tasks by title or description

-Sort tasks by Due Date, Priority, Creation Date, or Title

-Sort in Ascending or Descending order

-Dark Mode

-Toggle between light and dark themes

-User preference is saved in localStorage

-Statistics

-View total, completed, and pending tasks in the sidebar

Technologies Used

-HTML5

-CSS3 

-JavaScript

-LocalStorage for persistent data
